connect transport handler runs the Wayland protocol over transport (typically created with Unix_transport.of_socket). It spawns a background thread to handle incoming messages.

parameter trace

Used to trace all messages sent and received. The default tracer logs messages at debug level, and the log's source is set to debug level if $WAYLAND_DISPLAY is "1" or "server" the first time connect is called.
