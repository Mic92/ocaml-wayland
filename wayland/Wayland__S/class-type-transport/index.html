Class type Wayland__S.transport

A transport is used to send and receive bytes and file descriptors. Typically this will just call the usual Unix sendmsg and recvmsg functions, but other transports are possible.

method send : Cstruct.t -> Unix.file_descr list -> unit Lwt.t

send data fds transmits the bytes of data and the file descriptors in fds.

method recv : Cstruct.t -> (int * Unix.file_descr list) Lwt.t

recv buffer reads incoming data from the remote peer. The data is read into buffer and the method returns the number of bytes read and the list of attached file descriptors.
